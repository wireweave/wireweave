# Wireweave v2.0 — Beta

> Released: 2026-04-29 · Beta

Wireweave v2.0 introduces a built-in AI agent, frees up MCP tools across every plan, and ships the plugin marketplace. Plans, pricing, and credits stay identical to v1.

## Highlights

### § 1.1 Built-in AI agent

Generate and refine wireframes without bringing your own LLM key. Free users get a daily free quota; paid plans behave as before.

### § 1.2 MCP tools are free

Every MCP tool call now costs **0 credits** across plans. LLM-backed actions (`AI generate` / `AI improve`) still consume credits at the same rates as before.

### § 1.3 Plugin marketplace

Browse and install Skills / MCP extensions in one place. CLI, Cursor, Claude, and VSCode integrations live on the same page.

### § 1.4 Billing unchanged

| Plan       | Monthly | Yearly  | Monthly credits |
| ---------- | ------- | ------- | --------------- |
| Free       | $0      | —       | 1,000           |
| Basic      | $9      | $86     | 10,000          |
| Pro        | $29     | $278    | 50,000          |
| Enterprise | Contact | Contact | Unlimited       |

## What stays the same

- Plans (Free / Basic / Pro / Enterprise)
- Monthly and yearly prices, annual discount
- Monthly credits, overage rate, req-per-minute caps
- Wireframe data model (`.wf` / `.wireframe` still works as-is)
- Existing API keys — no rotation required

## Beta scope

- Some model responses may be slightly delayed during A/B comparison runs.
- Conversation history is stored server-side — opt-out under **Dashboard → Settings → AI consent**.
- A few marketplace cards link to external pages until install automation ships at GA.
- Beta announcement emails go only to people who opted in at signup.

## Migration guide

No action required for existing users. A few things worth checking:

1. **API keys** — still valid; no rotation needed.
2. **Billing** — same cycle and amount; Paddle receipts unchanged.
3. **AI consent** — a one-time consent prompt appears the first time you use the agent (GDPR/PIPA).
4. **MCP server** — existing configuration keeps working; tool calls auto-bill at 0 credits.

## Korean / 한국어

### 핵심 변화

1. **내장 AI 에이전트** — 자기 LLM 키 없이 와이어프레임 생성/개선이 가능합니다.
2. **MCP 도구 무료화** — 모든 MCP 도구 호출을 0 크레딧으로 변경했습니다.
3. **플러그인 마켓플레이스** — Skills / MCP 확장을 한 화면에서 둘러보고 설치할 수 있습니다.
4. **결제 정책 그대로** — 구독·가격·크레딧은 v1과 동일합니다.

### 변경 없음

- 구독 플랜 (Free / Basic / Pro / Enterprise)
- 월·연 가격 및 연간 할인율
- 월 크레딧 / 초과요금 / req-per-min 한도
- 와이어프레임 데이터 모델 (`.wf` / `.wireframe` 호환 그대로)
- 발급된 API 키 — 재발급 불필요

### 마이그레이션 가이드

기존 사용자는 별도 작업이 필요 없습니다.

1. **API 키** — 그대로 사용 가능합니다.
2. **결제** — 청구 주기/금액 변경 없음. Paddle 영수증도 동일하게 발송됩니다.
3. **AI 사용 동의** — 처음 에이전트 사용 시 한 번 동의 화면이 뜹니다 (GDPR/PIPA).
4. **MCP 서버** — 기존 설정 그대로. 도구 호출은 자동으로 0 크레딧 처리됩니다.
